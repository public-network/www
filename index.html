Preface

Decentralize and open source Humanity…

In a centralized world where power and control are abused and access is limited, the urgency to decentralize and open source Humanity has never been greater.

We, as Patrons, present Public Network, the Interstate of Humanity, a decentralized, open source network and permanent neutral ground in which to reside onchain.

Patrons protect, support, and evolve Humanity by contributing to initiatives that decentralize and open source identities, assets, information, transactions, and authentications.

If you, as an individual, group, or institution are committed to decentralizing and open sourcing Humanity, one initiative at a time, become a Patron.

Introduction

The world is dominated by centralized power structures that control not only digital lives but increasingly physical existence as well. A handful of platforms hold the keys to identity, assets, information, transactions, and authentication, shaping what is shared and owned while limiting access and stifling innovation.

Public Network is not an alternative but a radical reimagining of how people organize, interact, and thrive together. By redistributing power from the few to the many, Public Network creates an open, decentralized space where collaboration and transparency are the norm, and trust is restored through authentication and accountability.

In an era where artificial intelligence and autonomous agents increasingly create, act, and transact, the question of authorship becomes foundational. Who made this? Who signed this? Who authorized this? Public Network records the answer, permanently and neutrally. Whether a human, an institution, or an autonomous agent, every action carries the cryptographic signature of its source. Provenance becomes universal.

Public Network is a protocol, a brand, and an open source interface. Nothing more. Its permanence comes from its minimalism. Its neutrality comes from its ownerlessness. Its power comes from its Patrons.

Identity

Who someone is.

A self-owned identity that exists independently of any centralized authority. Cryptographically verifiable, permanently linked to the keys that created it, and usable anywhere on Public Network. An identity may represent an individual, group, institution, or autonomous agent. Any entity capable of holding a cryptographic key can hold an identity.

Identities on Public Network are ERC-721 tokens (PNI). Each identity owns a primary handle, a set of linked wallets, and an avatar. External accounts like Google, X, GitHub, and email can be linked to extend reputation across systems.

~/my-app · identity 
// Claim an identity handle
import { claimIdentity } from '@publicnetwork/sdk/identity';

const identity = await claimIdentity({
  handle: 'alice',
  wallet: signer,
});

// → PublicNetworkIdentity #1024 · 0xAlice…

Asset

What someone owns.

Digital representations of real-world, virtual, and intellectual resources: property, art, collectibles, digital creations, intellectual property. Assets on Public Network carry full ownership, transparent provenance, and composition. One asset can be composed of many others; every link is attested onchain.

Assets are ERC-721 tokens (PNA) with optional royalty routing via ERC-2981. Secondary sales route the protocol fee to the Public Network treasury automatically.

~/my-app · asset 
// Tokenize an asset with provenance
import { tokenizeAsset } from '@publicnetwork/sdk/asset';

const asset = await tokenizeAsset({
  identity,
  mediaUri: 'ar://bafy…',
  composedOf: [parentAssetId],
});

// → PublicNetworkAsset #4271 · composed

Information

What someone says.

A decentralized knowledge layer: posts, articles, scientific discoveries, news, provenance claims. Every piece of information is signed by its author's identity and recorded permanently. Signed, timestamped, and unforgeable.

Information lives inside Spaces (PNS), containers for threads, publications, or communities. A Space is addressable by name. Anyone can create one.

~/my-app · information 
// Create a signed post
import { post } from '@publicnetwork/sdk/information';

const signed = await post.sign({
  identity,
  body: 'Hello, Public Network.',
  space: 'announcements',
});

const pending = await post.createDirect(signed);
await pending.awaitConfirmation();

Transaction

What someone did.

A secure, decentralized record of value exchange: sales, listings, coin launches, cross-chain settlements. Every transaction carries the signatures of its participants and is recorded transparently, with fees routed at the protocol layer.

The protocol fee is fixed at 1% of transaction value. Hard-coded, non-configurable. It flows to the Public Network treasury and funds ongoing development, audits, and infrastructure. Apps built on top of Public Network can layer their own fee on top; the creator keeps the remainder.

~/my-app · transaction 
// Launch a coin with a 3-way fee split
import { launchCoin } from '@publicnetwork/sdk/transaction';

const coin = await launchCoin({
  identity,
  name: 'Example',
  symbol: 'EXMPL',
  totalFeeBps: 250,    // 2.50% total
  appCutBps: 50,        // 0.50% to the app
  // protocol takes 1.00% (fixed), creator keeps 1.00%
});

Authentication

What someone authorized.

A decentralized system where identities attach cryptographic signatures to approve actions, transactions, and information. Authentication turns a raw signature into a durable permission record: a sign-in, a delegation to an agent, an approval of a third-party app.

Delegation is first-class: an identity can authorize an autonomous agent to act on its behalf, revocable at any time. Every delegation is recorded; every revocation is recorded; the chain of authority is always walkable.

~/my-app · authentication 
// Delegate limited authority to an agent
import { delegation } from '@publicnetwork/sdk/authentication';

const d = await delegation.create({
  from: identity,
  to: agentKey,
  scope: ['post.create'],
  expires: '2026-12-31',
});

// Revoke any time
await delegation.revoke(d.uid);

Architecture

Public Network is built on Ethereum, the most neutral and decentralized programmable ledger in existence. It settles on Optimism, an Ethereum layer that provides fast, low-cost, permanent records.

Public Network does not run its own chain, does not require its own consensus, and does not issue its own currency. It inherits the security and neutrality of Ethereum itself.

Schemas

The five pillars are realized as 20 open source schemas on EAS, registered once on Optimism. Each schema is public, permissionless to read, and writable by the per-schema allowlist. Writers are curated at launch; reads are universal.

No participant can modify, remove, or gate these schemas. They are registered on Optimism and belong to Humanity.

Build on Public Network

Applications read and write the five pillars directly via the SDK. No permission, no registration, no endorsement. They are simply built.

zsh 
$ pnpm add @publicnetwork/sdk

$ cat > hello-pn.ts <<'EOF'
import { configure } from '@publicnetwork/sdk';
import { claimIdentity } from '@publicnetwork/sdk/identity';
import { post } from '@publicnetwork/sdk/information';

configure({ network: 'optimism' });

const id = await claimIdentity({ handle: 'alice', wallet });
await post.createDirect(
  await post.sign({ identity: id, body: 'hello' })
);
EOF

$ pnpm tsx hello-pn.ts

Five modules mirror the five pillars: identity, asset, information, transaction, authentication. Core plumbing including EIP-712 signing, relay failover, and pending-attestation confirmation sits underneath.

Sign in with Public Network

A decentralized single sign-on primitive. An identity that works across every app built on Public Network. Wallet-first, portable, revocable.

The flow is simple: the user’s wallet signs a scoped authorization payload; the app receives a signed attestation that names the identity, the requested scopes, and the expiration. No OAuth server. No Facebook. No Google.

~/my-app · sign-in 
// Request sign-in from a user
import { authentication } from '@publicnetwork/sdk/authentication';

const session = await authentication.signIn({
  app: 'my-app.example',
  scopes: ['identity.read', 'post.create'],
});

// session.identity · session.expires · session.signature

The Perfection of the Imperfect Circle

Feeling insecure and alienated from others, the Imperfect Circle tried to join the ranks of the perfect circles, but over time its true self started to diminish along with the traces of its imperfect shape.

After the long quest for an ideally manufactured line, the Imperfect Circle realized that perfection was an abstraction and utopia, an elusive ideal with many interpretations that was impossible to achieve. Eventually, the purest form of perfection was found in its stead: authenticity with all its flaws and mistakes. The Imperfect Circle came back to its innermost being, its original form, unique, real, and free, accepting the innate beauty of itself and others.

But its imperfect shape can only be seen and appreciated by those who are able to look deeper and recognize its true form.
